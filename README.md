# Blog Platform with Comments

## Technologies

- HTML
- CSS
- JavaScript
- Python (Flask)
- SQLite

## Features

- User Registration
- User Login
- Authentication
- Create Blog Posts
- Edit Blog Posts
- Delete Blog Posts
- Comment System
- SQLite Database
- Responsive Design

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python app.py
```

Open your browser:

```
http://127.0.0.1:5000
```
